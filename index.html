Privacy Policy

Effective Date: 2/26/2025

1. Introduction

This privacy policy explains how Burnout Tracker ("we," "our," or "us") collects, uses, and protects user data. We are committed to ensuring user privacy and data security while providing a seamless experience.

2. Information We Collect

We collect only the minimum data necessary to provide and improve the functionality of the extension.

  • Web History: We analyze time spent on websites to generate engagement insights.
  • User Activity: We monitor time spent on active tabs and idle periods to improve productivity tracking.
  • Website Content: We may analyze page metadata but do not store page contents.

3. How We Use the Data

The collected data is used to provide insights into time management, enhance user productivity, and improve overall extension performance.

4. Data Security

We implement industry-standard security measures to protect stored data. No data is shared or sold to third parties.

5. User Controls

Users can manage or delete stored data at any time via extension settings.
